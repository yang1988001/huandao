## 🚀 GitHub Pages 部署指南

### 第一步：创建 GitHub 账号
1. 访问 https://github.com
2. 点击右上角 "Sign up" 注册
3. 验证邮箱

### 第二步：创建新仓库
1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 仓库名称：`spring-outing-2026`（或任意名称）
4. 选择 "Public"（公开）
5. **不要**勾选 "Add a README file"
6. 点击 "Create repository"

### 第三步：上传文件
#### 方法A：直接上传（推荐）
1. 在新仓库页面点击 "uploading an existing file"
2. 将本文件夹中的所有文件（11张图片 + index.html）拖放到上传区域
3. 点击 "Commit changes"

#### 方法B：使用 Git 命令
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/spring-outing-2026.git
git push -u origin main
```

### 第四步：开启 GitHub Pages
1. 点击仓库顶部的 "Settings" 标签
2. 左侧菜单找到 "Pages"
3. "Source" 部分选择 "Deploy from a branch"
4. "Branch" 选择 "main"，文件夹选择 "/ (root)"
5. 点击 "Save"

### 第五步：获取链接
- 等待 1-2 分钟部署完成
- 访问链接：`https://你的用户名.github.io/spring-outing-2026/`
- 将此链接分享到微信即可
